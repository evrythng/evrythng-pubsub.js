Setup

1. Set the 'operator' or 'accessToken' query parameter.

2. Open devtools to use the 'operator' or 'accessToken' objects.

3. Set the 'apiUrl' parameter to override the default MQTT/WS URL.


Example

Subscribe to all actions:

operator.action('all').subscribe(console.log);

Then publish an action to test the subscription:

operator.action('all').publish({ type: '_MyCustomType' });
